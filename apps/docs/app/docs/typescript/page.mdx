# TypeScript

## Getting Started

To get started with **Sign in with Aptos (SIWA)** in your TypeScript application, Aptos Labs provides a number of libraries and integrations to help you get started.

### [@aptos-labs/siwa](/docs/typescript/utilities)

The `@aptos-labs/siwa` library is a TypeScript library that provides a set of utilities for working with SIWA. These utilities are designed to help you construct SIWA messages and verify SIWA messages.
